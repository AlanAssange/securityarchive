# NTP

NTP (Network Time Protocol) es un protocol utilizado para sincronizar los relojes de las computadoras y dispositivos de una red. Asegura que todos los sistemas mantengan el tiempo de forma acertada y coordinada. NTP opera en el puerto UDP 123 y utiliza algoritmos para ajustar el reloj, proveyendo una certeza de milisegundos.
